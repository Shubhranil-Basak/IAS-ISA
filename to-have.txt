more on IAS ISA assembly language
1) comments are defined using '//' operator and the entire line from that will be ignored.
2) the assembly program starts form the beginning of the file.
3) each line can contain up to two instructions separated by a ";".
4) the code will be the same as it is stored in memory and will look something like this.

// IAS ISA assembly language

<instruction> M(<memory>, <optional parameters>) ; <instruction> M(<memory>, <optional parameters>)
... // comments
...
